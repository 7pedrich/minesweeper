Minesweeper game I made for fun

Controls:
- Touch with your character to Reveal cell 
- LMB to Toggle flag on/off
This is how it should look like when implemented:

![Demo (gif)](https://github.com/user-attachments/assets/6b309761-6411-4be8-a046-53e395c3d6f9)

You can also customize certain things like:
- Bomb rate percent
- Grid size X
- Grid size Y
- Text colors for nearby bomb count
- Cell colors (in the `Minesweeper:_createPart()` function, at the Part.Color line.)
